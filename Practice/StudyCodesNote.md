- **cout** = character output 

- **endl**: new line / something like \n .  
 Also flushes the output buffer, ensuring that all output is sent to the console immediately.
This can be useful in certain situations where you want to ensure that the output is displayed before any further processing occurs.


- ( **return0;** )  : signals the OS that the program finished successffully. After this line of code if u write other codes, it wont work. its like a finish line from you that says no more code is written.

- **const**  variable means, u can not change the value of that variable, it is constant.
for example we put pi = 3.14, so in future code u cant change the pi value.

**What is a pointer?**
A pointer is a variable that stores a memory address.   
Instead of holding a number like int x = 5, it holds the location where a value is stored.


The pointer is like the map.
The * is like the pin on the map showing the exact spot.
Then *pointer is like going to that pin and taking the treasure.
